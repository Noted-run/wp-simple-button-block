Often when external links are listed, there is no OG image to link to; Simple Button Block provides a simple button block where you simply enter an image, title, and link.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/static` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
